# Какой тебе подходит нейрообраз?

Интерактивный квиз (одна страница `index.html`). После выгрузки на **GitHub Pages** появится публичная ссылка вида:

`https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

---

## 1. Создать репозиторий на GitHub

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт.
2. **New repository** (новый репозиторий).
3. Имя на латинице, например: `neuro-obraz-quiz` (без пробелов удобнее для ссылки).
4. Можно выбрать **Public**.
5. **Не** добавляйте README из шаблона, если уже есть файлы в папке проекта — или добавьте, потом разберёте конфликт.
6. Нажмите **Create repository**.

---

## 2. Загрузить файлы с компьютера

### Вариант A: сайт GitHub (проще всего)

1. Откройте страницу созданного репозитория.
2. **Add file** → **Upload files**.
3. Перетащите в окно файлы из этой папки: `index.html`, `README.md`, `.gitignore`, `.gitattributes` (если есть).
4. **Commit changes**.

### Вариант B: Git в командной строке

Установите [Git for Windows](https://git-scm.com/download/win), откройте терминал в папке проекта и выполните (подставьте свой логин и имя репозитория):

```bash
git init
git add .
git commit -m "Первый коммит: квиз нейрообраз"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПО.git
git push -u origin main
```

При первом `push` GitHub попросит войти (браузер или токен).

---

## 3. Включить публичную ссылку (GitHub Pages)

1. В репозитории: **Settings** → слева **Pages**.
2. В блоке **Build and deployment**:
   - **Source**: *Deploy from a branch*.
   - **Branch**: `main`, папка **`/` (root)**.
3. **Save**.
4. Через 1–3 минуты на той же странице **Pages** появится ссылка на сайт — ей можно делиться.

Если `index.html` лежит в корне репозитория, откроется сразу главная страница квиза.

---

## 4. Обновления после правок

Снова загрузите изменённый `index.html` через **Upload** или сделайте `git add`, `git commit`, `git push` — страница обновится через минуту-две.

---

## Зависимости

Сайт подгружает шрифты и фото с **Google Fonts** и **Unsplash** — нужен интернет у того, кто открывает ссылку.
